# Installation & Administration Overview

Welcome to Prediction Guard's installation and administration documentation. This section covers everything you need to install, deploy, and manage your Prediction Guard instance.

## What is Prediction Guard?

Prediction Guard allows you to:

- **Deploy any model** you want through our admin panel
- **Host on-premises** or on any cloud provider
- **Maintain air-gapped** environments for maximum security
- **Manage your own infrastructure** with full control
- **Use any open model** from Hugging Face or other sources

## Installation Options

### Cloud Installation
Install on major cloud providers with managed services:
- [AWS](./aws.mdx) - Amazon Web Services
- [Azure](./azure.mdx) - Microsoft Azure  
- [GCP](./gcp.mdx) - Google Cloud Platform

### On-Premises Installation
Install in your own data center:
- [Docker](./docker.mdx) - Container-based installation
- [Kubernetes](./kubernetes.mdx) - Orchestrated container installation

### Air Gapped Installation
Install in isolated environments:
- [Air Gapped](./air-gapped.mdx) - Offline installation guide

## Admin Panel Features

Our admin panel provides:

- **Model Management** - Deploy and configure any model
- **API Key Management** - Control access and permissions
- **Monitoring & Analytics** - Track usage and performance
- **Configuration Management** - Customize your deployment

## Getting Started

1. **Choose your installation method** from the Installation section above
2. **Follow the specific installation guide** for your environment
3. **Access the admin panel** to configure models and API keys
4. **Set up monitoring and maintenance** procedures
5. **Start using your Prediction Guard instance**
